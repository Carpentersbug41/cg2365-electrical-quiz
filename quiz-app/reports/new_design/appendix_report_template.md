# AI Improvement Report Template Appendix

Use this template for each formal analysis cycle.

## Metadata
- report id
- period start
- period end
- runtime version
- variant id
- baseline variant id
- sample size
- included lessons
- data-quality notes

## Required Sections
- executive summary
- runtime summary
- chunk-level analysis
- learner-message analysis
- fatigue analysis
- repair and misconception analysis
- microbreak analysis
- learning outcome analysis
- support dependence analysis when available
- transfer analysis when available
- recovery capacity analysis when available
- pace mismatch analysis when available
- learning efficiency analysis when available
- AI-change quality analysis when available

## Required Issue Format
For each issue include:
- problem
- evidence
- likely cause
- research basis
- proposed change
- change type
- risk level
- expected effect
- metric to watch
- rollback condition
